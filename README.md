# Casio Style Calculator

A simple **Casio-style calculator** built using **HTML, CSS, and
JavaScript**.\
This project recreates the look and feel of a traditional **Casio
calculator**, allowing users to perform basic arithmetic operations
directly in the browser.

------------------------------------------------------------------------

## Project Overview

The Casio Calculator is a web-based calculator designed to simulate the
appearance and functionality of a classic handheld calculator.\
It includes a digital display, responsive buttons, and supports common
mathematical operations.

This project was created to practice **front-end development skills**,
focusing on **user interface design and JavaScript functionality**.

------------------------------------------------------------------------

## Features

-   Casio-style calculator interface
-   Digital 7‑segment display font
-   Basic arithmetic operations:
    -   Addition (+)
    -   Subtraction (-)
    -   Multiplication (×)
    -   Division (÷)
-   Parentheses support `( )`
-   Decimal number calculations
-   Delete button (DEL)
-   Clear button (AC)
-   Responsive button interaction
-   Error handling for invalid calculations
-   Displays copyright and last modified date

------------------------------------------------------------------------

## Technologies Used

-   **HTML5** -- Structure of the calculator
-   **CSS3** -- Styling and layout
-   **JavaScript** -- Calculator logic and functionality
-   **Segment7 Font** -- Digital calculator display style

## How It Works

-   Button clicks send numbers or operators to the display.
-   The display updates dynamically using JavaScript.
-   When the **equals (=)** button is pressed, the expression is
    evaluated using JavaScript.
-   The calculator handles errors and resets the display if an invalid
    expression occurs.
